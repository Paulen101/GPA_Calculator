# FHSU GPA Calculator

A React + Vite app for estimating Fort Hays State University semester GPA, cumulative GPA, and weighted class grades.

## Features

- Semester GPA calculator with repeated-course exclusion and what-if grades
- Cumulative GPA projection using previous GPA and earned credits
- Weighted class grade calculator
- Final exam target calculator based on the live assignment average

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

This project is an unofficial calculator and should not be treated as an official academic record. Confirm final GPA and standing through FHSU systems such as Workday and the Registrar.
